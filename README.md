## Описание

Этот проект представляет собой телеграм-бота, который позволяет пользователям решать шахматные задачи в интерактивном режиме. Пользователи могут решать задачи как гости, не входя в свой аккаунт Lichess, либо войти в свой аккаунт Lichess для решения задач с учетом их рейтинга.

## Стек технологий

- Python 3.8+
- `python-telegram-bot` (библиотека для работы с Telegram API)
- `python-chess` (библиотека для работы с шахматными досками)
- `lichess_client` (библиотека для работы с API Lichess)

## Базовый функционал

- Отправка шахматной задачи пользователю.
- Получение ходов от пользователя и проверка их на корректность.
- Отправка обратной связи пользователю после каждого хода.
- Возможность решения задач как гостем, без входа в аккаунт Lichess.
- Возможность входа в аккаунт Lichess и решения задач с учетом рейтинга пользователя.

## Примерная архитектура и описание компонентов

- **ChessBot**: Главный класс, отвечающий за запуск бота и обработку входящих запросов от пользователей. Он содержит методы для отправки задач, обработки ввода пользователей и отправки обратной связи.

- **ChessPuzzle**: Класс, представляющий шахматную задачу. Он содержит информацию о текущей позиции на доске, цели задачи и другие параметры. Также в этом классе реализованы методы для генерации изображения доски и проверки ввода пользователей.

- **ChessBoard**: Класс, представляющий шахматную доску. Он содержит информацию о расположении фигур на доске и методы для изменения состояния доски в соответствии с ходами игроков.

- **ChessImageGenerator**: Класс, отвечающий за генерацию изображений досок шахмат с помощью библиотеки python-chess.

- **TelegramAPI**: Класс, обеспечивающий взаимодействие с API Telegram для отправки и получения сообщений от пользователей.

- **LichessAPI**: Класс, обеспечивающий взаимодействие с API Lichess для аутентификации пользователей, получения задач и информации о рейтинге.
